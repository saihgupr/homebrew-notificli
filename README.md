# homebrew-notificli

Official Homebrew tap for [NotifiCLI](https://github.com/saihgupr/NotifiCLI), a lightweight, headless macOS command-line tool for sending actionable, persistent notifications.

## Installation

To tap this repository:

```bash
brew tap saihgupr/notificli
```

Then install the `notificli` cask:

```bash
brew install --cask notificli
```

## Documentation

For full usage instructions and flags, see the main [NotifiCLI README](https://github.com/saihgupr/NotifiCLI#readme).
